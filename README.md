simple lara installer.


step 1:
create in server root dir one folder: "lara_server"

step 2:
download "lara_server" package from git into lara_server

step 3:
run program.
and follow the description.

how to run:
```bash
$ ./installer.sh
```
or (for example on easyname-server)

```bash
$ bash installer.sh
```
